## 开发

本项目使用 [Docusaurus](https://docusaurus.io/docs) 开发。

```sh
yarn install
yarn start
```

## 部署

- 推送到 `master` 的改动，会自动发布到 Github Page。
- 手动部署

  ```sh
  GIT_USER=<Your GitHub username> yarn deploy
  ```
